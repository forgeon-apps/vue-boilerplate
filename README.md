# Vue.js

This directory is a brief example of a [Vue.js](https://vuejs.org/) app that can be deployed to Forgeon with zero configuration.

## Deploy Your Own

Deploy your own Vue.js project with Forgeon.

[![Deploy with Forgeon](https://forgeon.io/button)](https://forgeon.io/new/clone?repository-url=https://github.com/forgeon-apps/examples/tree/main/framework-boilerplates/vue&template=vue)

_Live Example: https://vue-template.forgeon.io_

## Running Locally

```
yarn install
```

#### Compile and hot-reload for development

```
yarn serve
```

#### Compile and minify for production

```
yarn build
```

#### Lint and fix files

```
yarn lint
```
